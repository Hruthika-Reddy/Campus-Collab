Basic Code Examples

C Programming

// Factorial using recursion
int factorial(int n) {
    if(n == 0) return 1;
    return n * factorial(n-1);
}
// Linked List Implementation
struct Node {
    int data;
    struct Node* next;
};

void insert(struct Node** head, int data) {
    struct Node* newNode = (struct Node*)malloc(sizeof(struct Node));
    newNode->data = data;
    newNode->next = *head;
    *head = newNode;
}

Python

# Fibonacci series
def fibonacci(n):
    a, b = 0, 1
    for _ in range(n):
        print(a, end=' ')
        a, b = b, a+b

Important Questions & Answers

C Programming

  • Q: What is pointer in C?
    A: A pointer is a variable that stores the memory address of another variable.
  • Q: Difference between call by value and call by reference?
    A: Call by value passes a copy of the variable, while call by reference passes the address.
  • Q: What is recursion?
    A: A function calling itself directly or indirectly is called recursion.
  • Q: What is structure in C?
    A: A structure is a user-defined data type that groups related data items of different types.

Data Structures

  • Q: What is a linked list?
    A: A linear data structure where elements are linked using pointers.
  • Q: Explain stack and queue.
    A: Stack is LIFO (Last In First Out) while queue is FIFO (First In First Out).
  • Q: What is binary search tree?
    A: A tree data structure where each node has at most two children with left child < parent < right child.
  • Q: Explain graph traversal algorithms.
    A: BFS (Breadth First Search) and DFS (Depth First Search) are two main graph traversal algorithms.

Operating Systems

  • Q: What is virtual memory?
    A: A memory management technique that gives an illusion of a large main memory.
  • Q: Explain process scheduling algorithms.
    A: FCFS, SJF, Round Robin, Priority Scheduling are common CPU scheduling algorithms.
  • Q: What is deadlock?
    A: A situation where two or more processes are waiting indefinitely for resources held by each other.
  • Q: What is paging?
    A: A memory management scheme that eliminates the need for contiguous allocation of physical memory.
