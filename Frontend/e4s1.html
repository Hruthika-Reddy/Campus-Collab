Elective IV & V Pool - Detailed Course Guide

This comprehensive guide provides detailed information about all Professional Elective Courses (PEC) available under Elective IV & V Pool. Each course description includes:

  • Complete course code and title information

Unix Shell Programming

23CS4121
Category: PEC
Credits: 3 (3-0-0)
L-T-P: 3-0-0

Course Learning Objectives

This course aims to:

  1. Develop written technical communication skills and effective use of concepts and terminology
  2. Provide facility with UNIX command syntax and semantics
  3. Develop ability to read and understand specifications, scripts and programs
  4. Enhance individual capability in problem solving using UNIX tools
  5. Enable students to demonstrate mastery of course materials through discussions

Course Content

Unit I – Introduction to UNIX
8 Contact Hours

Architecture of Unix, Features of Unix, Unix Commands – PATH, man, echo, printf, script, passwd, uname, who, date, stty, pwd, cd, mkdir, rmdir, ls, cp, mv, rm, cat, more, wc, lp, od, tar, gzip, cpio, ln.

UNIX Utilities: Process utilities, disk utilities, networking commands, unlink, du, df, mount, umount, find, unmask, ulimit, ps, w, finger, arp, ftp, telnet, rlogin. Text processing utilities and backup utilities, detailed commands to be covered are tail, head, sort, uniq, grep, egrep, fgrep, cut, paste, join, tee, pg, comm, cmp, diff, tr, awk.

Unit II - File Management
6 Contact Hours

Introduction to unix file system, vi editor, file handling utilities, security by file permissions, File Structures, System Calls for File Management – create, open, close, read, write, lseek, link, symlink, unlink, stat, fstat, lstat, chmod, chown, Directory API – opendir, readdir, closedir, mkdir, rmdir, umask.

Learning Resources

Text Books

  • Behrouz A. Forouzan & Richard F. Gilberg, "Unix and Shell Programming", 1st edition, Cengage Learning, 2003

Reference Books

  • Sumitabha Das, "Your Unix: The Ultimate Guide", 1st edition, Tata Mcgraw hill, 2001
  • Graham Glass, King Ables, "Unix for programmers and Users", 3rd edition, Pearson Education, 2003
  • Kernighan & Pike, "The UNIX Programming Environment", 1st edition, Pearson Education India, 2015

Course Outcomes

At the end of the course, students will be able to:

  1. Demonstrate good documentation organization and readability
  2. Perform data organization, problem solving and research
  3. Create simple effective user interfaces
  4. Apply structured programming effectively
  5. Produce printed output demonstrating completion of test plans

Assessment Method

Assessment Tool Weightage (%)
Weekly tests 10%
Monthly tests 30%
End Semester Test 60%
Total 100%

Deep Learning

23CS4122
Category: PEC
Credits: 3 (3-0-0)
L-T-P: 3-0-0

Course Objectives

  1. To build the foundation of deep learning
  2. To understand how to build the neural network
  3. To enable the students to develop successful machine learning concepts

Course Content

UNIT - I
8 Contact Hours

Introduction: Feed forward Neural networks, Gradient descent and the back propagation algorithm, Unit saturation, the vanishing gradient problem, and ways to mitigate it. RelU Heuristics for avoiding bad local minima, Heuristics for faster training, Nestors accelerated gradient descent, Regularization, Dropout.

UNIT - II
7 Contact Hours

Convolutional Neural Networks: Architectures, convolution/pooling layers, Recurrent Neural Networks: LSTM, GRU, Encoder Decoder architectures.

Learning Resources

Text Books

  • Deep Learning by Ian Goodfellow, Yoshua Bengio and Aaron Courville, MIT Press.
  • The Elements of Statistical Learning by T. Hastie, R. Tibshirani, and J. Friedman, Springer.

Reference Books

  • Probabilistic Graphical Models by D. Koller, and N. Friedman, MIT Press.

Course Outcomes

Upon successful completion of the course, students will be able to:

  1. Learn the fundamental principles of deep learning
  2. Identify the deep learning algorithms for various types of learning tasks in various domains
  3. Implement deep learning algorithms and solve real-world problems

Assessment Method

Assessment Tool Weightage (%)
Weekly tests 10%
Monthly tests 30%
End Semester Test 60%
Total 100%

VLSI

23CS4123
Category: PEC
Credits: 3 (3-0-0)
L-T-P: 3-0-0

Course Learning Objectives

  1. To make students understand the introductory concepts of Very Large Scale Integrated Circuits design
  2. To cover HDLs, CMOS circuit design, Layout techniques and fabrication

Course Content

Unit I
13 Contact Hours

Finite State Machines and Verilog HDL: Mealy state machines, Moore state machines, Conversion of Mealy machines to Moore machines and vice-versa, Redundancy elimination, Sequence detectors with and without overlapping, Parity detectors and generators.

Verilog HDL: Gate Level modeling, Dataflow modeling, Behavioral modeling, Delay modeling, Writing test bench, FSM modeling.

Unit-II
7 Contact Hours

Digital System Design and Verilog HDL: Specifications, Moore's Law, Data path design, Control path design, GCD Datapath design, GCD Control path design, State machine for traffic light controller. FPGA Design flow, ASIC Design flow. Verilog HDL: Datapath, Control path, System Design

Learning Resources

Text Books

  • Stephen Brown, Zvoko Vranesic, 'Fundamentals of Digital Design using Verilog', Mc Graw Hill publications
  • Douglas A.Pucknell and Kamran Eshraghian, 'Basic VLSI Design', PH publications.

Reference Books

  • Samir Palnitkar, 'Verilog HDL - A Guide to Digital Design and Synthesis', Pearson publications.
  • Stephen D. Senturia, 'Microsystem Design', Kluwer Academic Publishers.

Course Outcomes

At the end of the course, students will be able to:

  1. Understand specifications of VLSI designs, Moore's Law
  2. Differentiate VLSI Design flows - FPGA, ASIC
  3. Understand the concepts of Finite State Machines and its relevance in IC Design
  4. Model digital designs using hardware description language
  5. Understand the different steps involved in IC fabrication process
  6. Understand the concept of IC layout
