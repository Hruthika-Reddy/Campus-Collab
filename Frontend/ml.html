23CS3202 - Machine Learning

Course Category: PCC
Credits: 4
L-T-P: 3-1-0

Course Learning Objectives

Understanding ML Foundations

Grasp the fundamental theory and mathematical underpinnings of machine learning algorithms.

Problem Formulation

Learn to frame real-world problems as machine learning tasks with appropriate formulations.

Algorithmic Knowledge

Understand a wide range of ML algorithms, their assumptions, strengths, and limitations.

Practical Application

Develop skills to implement ML solutions for problems of moderate complexity.

Machine Learning Workflow

  1. Problem Definition: Identify the task (classification, regression, clustering, etc.)
  2. Data Collection: Gather relevant data from various sources
  3. Data Preprocessing: Clean, normalize, and transform data
  4. Feature Engineering: Select and create meaningful features
  5. Model Selection: Choose appropriate algorithm(s)
  6. Training: Fit model to training data
  7. Evaluation: Assess performance on test data
  8. Optimization: Tune hyperparameters
  9. Deployment: Implement model in production
Problem → Data → Model → Evaluation

Unit I: Introduction (10 Contact Hours)

Learning Problems and Perspectives

Overview of different types of learning problems and key issues in machine learning.

Learning Paradigms

  • Supervised Learning: Learning from labeled examples
  • Unsupervised Learning: Finding patterns in unlabeled data
  • Reinforcement Learning: Learning through rewards and penalties

Concept Learning

Understanding how machines learn concepts from examples using version spaces and candidate elimination algorithms.

Decision Tree Learning

Tree-based models that learn decision rules from data features.

Decision Tree Example

Decision trees work by recursively partitioning the feature space based on attribute values that maximize information gain.

Unit II: Neural Networks and Genetic Algorithms (12 Contact Hours)

Neural Network Fundamentals

Biological inspiration, artificial neuron model, network architectures.

Perceptrons

Single-layer neural networks and their limitations.

Multilayer Networks and Backpropagation

Deep networks trained using gradient descent and error backpropagation.

Weight Update Rule: wij ← wij + ηδjxi

Genetic Algorithms

Evolutionary computation inspired by biological evolution.

Genetic Algorithm Steps

  1. Initialization: Create random population of solutions
  2. Evaluation: Assess fitness of each solution
  3. Selection: Choose parents based on fitness
  4. Crossover: Combine parents to create offspring
  5. Mutation: Randomly modify some offspring
  6. Replacement: Form new population
  7. Termination: Stop when criteria met

Unit III: Bayesian Learning (9 Contact Hours)

Bayes Theorem

Foundation of probabilistic reasoning in machine learning.

P(A|B) = P(B|A)P(A) / P(B)

Bayesian Classifiers

  • Naïve Bayes Classifier
  • Bayes Optimal Classifier
  • Gibbs Algorithm

Bayesian Networks

Graphical models representing probabilistic relationships among variables.

EM Algorithm

Expectation-Maximization for learning with incomplete data.

Unit IV: Computational Learning (10 Contact Hours)

Learning Theory

Sample complexity, mistake bound model, PAC learning framework.

Rule-Based Learning

Sequential covering algorithms for learning sets of rules.

First-Order Rules

Learning rules with variables and predicates.

Unit V: Instance Based Learning (9 Contact Hours)

k-Nearest Neighbors

Non-parametric method that classifies based on majority vote of nearest neighbors.

Locally Weighted Regression

Memory-based method that approximates the target function locally.

Case-Based Learning

Problem-solving paradigm that uses specific past experiences.

Unit VI: Advanced Learning (10 Contact Hours)

Support Vector Machines

Maximum margin classifiers using kernel methods.

Minimize: ½||w||² + CΣξi
Subject to: yi(w·xi + b) ≥ 1-ξi, ξi ≥ 0

Reinforcement Learning

Q-learning and temporal difference learning for sequential decision making.

Q-Learning Update Rule

Q(s,a) ← Q(s,a) + α[r + γmaxa'Q(s',a') - Q(s,a)]

Where:

  • α = learning rate
  • γ = discount factor
  • r = immediate reward
  • s' = next state
  • a' = next action
