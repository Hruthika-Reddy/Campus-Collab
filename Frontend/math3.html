🎲 Probability Basics

Probability is a measure of the likelihood that an event will occur, quantified as a number between 0 (impossible) and 1 (certain).

Fundamental Concepts

1. Sample Space (S)

The set of all possible outcomes of an experiment.

Example: When rolling a die:

S = {1, 2, 3, 4, 5, 6}

2. Events

A subset of the sample space.

Example: Event A = "Rolling an even number"

A = {2, 4, 6}

3. Probability Rules

P(A) = Number of favorable outcomes / Total number of possible outcomes

Example: Probability of rolling a 3 on a fair die:

P(3) = 1/6 ≈ 0.1667

Probability Rules and Theorems

Addition Rule

P(A ∪ B) = P(A) + P(B) - P(A ∩ B)

For mutually exclusive events: P(A ∩ B) = 0

Multiplication Rule

P(A ∩ B) = P(A) × P(B|A)

For independent events: P(B|A) = P(B)

Bayes' Theorem

P(A|B) = [P(B|A) × P(A)] / P(B)

Medical Test Example

Suppose:

  • Disease prevalence P(D) = 1%
  • Test sensitivity P(T+|D) = 99%
  • Test specificity P(T-|¬D) = 95%

Probability of having disease given positive test:

P(D|T+) = [P(T+|D)P(D)] / [P(T+|D)P(D) + P(T+|¬D)P(¬D)] = (0.99×0.01)/(0.99×0.01 + 0.05×0.99) ≈ 16.7%

📊 Random Variables

A random variable is a variable whose possible values are numerical outcomes of a random phenomenon.

Types of Random Variables

Type Description Examples
Discrete Countable number of distinct values Number of customers, dice rolls
Continuous Uncountable values within an interval Height, weight, time

Probability Mass Function (PMF)

For discrete random variables:

p(x) = P(X = x)

Probability Density Function (PDF)

For continuous random variables:

P(a ≤ X ≤ b) = ∫ab f(x) dx

Expected Value and Variance

Expected Value (Mean)

E[X] = Σ x·p(x) (Discrete)
E[X] = ∫ x·f(x) dx (Continuous)

Variance

Var(X) = E[(X - μ)2] = E[X2] - (E[X])2

Standard Deviation

σ = √Var(X)

Dice Roll Example

For a fair six-sided die:

E[X] = (1+2+3+4+5+6)/6 = 3.5
Var(X) = [(1-3.5)² + (2-3.5)² + ... + (6-3.5)²]/6 ≈ 2.92
σ ≈ √2.92 ≈ 1.71

📈 Probability Distributions

Discrete Distributions

Binomial Distribution

Models number of successes in n independent trials

P(X=k) = C(n,k) pk(1-p)n-k

Where C(n,k) is the combination formula

Example: Probability of 3 heads in 5 coin flips (p=0.5):

P(X=3) = C(5,3) (0.5)3(0.5)2 = 10 × 0.125 × 0.25 = 0.3125

Poisson Distribution

Models rare events over time/space

P(X=k) = (e-λ λk) / k!

Continuous Distributions

Normal Distribution

The "bell curve" with mean μ and variance σ²

f(x) = (1/√(2πσ²)) e-(x-μ)²/(2σ²)

Standard normal distribution (μ=0, σ=1)

Central Limit Theorem

The sum of a large number of independent random variables is approximately normally distributed.

📉 Descriptive Statistics

Measures of Central Tendency

Measure Formula When to Use
Mean μ = (Σx)/n Normal distributions
Median Middle value Skewed distributions
Mode Most frequent value Categorical data

Example Dataset: 3, 5, 7, 7, 8, 10, 12

  • Mean = (3+5+7+7+8+10+12)/7 ≈ 7.43
  • Median = 7 (4th value)
  • Mode = 7 (appears twice)

Measures of Dispersion

Measure Formula Interpretation
Range Max - Min Total spread
Variance σ² = Σ(x-μ)²/n Squared deviations
Standard Deviation σ = √σ² Typical deviation
IQR Q3 - Q1 Middle 50% spread
