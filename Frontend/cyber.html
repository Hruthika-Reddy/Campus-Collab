About the Summit

The CyberSec Summit 2024 is an elite cybersecurity competition where participants tackle real-world security challenges across multiple domains. This event brings together the brightest minds to test their skills in penetration testing, secure system design, and threat analysis.

With cyber threats evolving rapidly, this summit provides a platform to develop practical defense skills through hands-on challenges guided by industry experts.

Competition Rules

To ensure fair play and maintain the integrity of the competition, all participants must adhere to the following rules:

Ethical Conduct

All activities must comply with ethical hacking principles. No real-world systems may be targeted outside the competition environment.

Team Composition

Teams of 2-4 members from the same institution. No cross-institution teams allowed. Team changes prohibited after registration.

Tools & Resources

Only pre-approved tools allowed. Internet access restricted to documentation only. No pre-built exploits or automated tools.

Flag Submission

Flags must be submitted in the specified format within the time limit. Multiple incorrect submissions may result in point penalties.

Scoring System

Dynamic scoring based on challenge difficulty and solve speed. First blood bonuses awarded for being first to solve a challenge.

Disqualification

Cheating, DoS attacks on infrastructure, or sharing solutions will result in immediate disqualification and potential bans.

Challenge Categories

Web Application Security

Medium-Hard

Exploit vulnerabilities in modern web applications including XSS, SQLi, CSRF, and authentication flaws. Challenges mimic real-world bug bounty scenarios.

Binary Exploitation

Hard

Reverse engineer compiled binaries to find and exploit memory corruption vulnerabilities like buffer overflows and ROP chains.

Cryptography

Medium

Break flawed cryptographic implementations, solve complex mathematical problems, and crack encryption schemes.

Network Security

Easy-Medium

Analyze network traffic, exploit protocol vulnerabilities, and penetrate secured network configurations.

Forensics

Easy-Hard

Recover hidden data from disk images, memory dumps, and network captures using forensic tools and techniques.

Secure System Design

Hard

Design and implement secure systems with proper authentication, authorization, and encryption mechanisms.
