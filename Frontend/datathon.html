About the Challenge

The Data Science Challenge 2024 is a premier competition where participants solve real-world problems using cutting-edge data science techniques. This event brings together the brightest analytical minds to test their skills in machine learning, statistical modeling, and data visualization.

In our data-driven world, this challenge provides a platform to develop practical data science skills through hands-on projects guided by industry leaders and academic experts.

Competition Rules

To ensure fair competition and maintain academic integrity, all participants must follow these guidelines:

Data Usage

Only provided datasets may be used. External data sources are prohibited unless explicitly allowed for specific challenges.

Team Composition

Teams of 1-3 members from the same institution. Cross-institution collaboration is not permitted.

Tools & Libraries

Open-source tools and libraries are encouraged. Commercial software requires prior approval.

Submission Timing

Solutions must be submitted before deadlines. Late submissions will receive point deductions.

Evaluation Criteria

Solutions judged on accuracy, innovation, reproducibility, and clarity of methodology.

Original Work

All submissions must be original work created during the competition period.

Challenge Tracks

Predictive Modeling

Advanced

Build accurate predictive models using structured datasets. Evaluate performance on hidden test sets with strict computational constraints.

Natural Language Processing

Intermediate-Advanced

Solve text classification, sentiment analysis, or generation tasks using modern NLP techniques and transformer architectures.

Computer Vision

Intermediate

Develop image classification, object detection, or segmentation models with provided image datasets and evaluation metrics.

Time Series Analysis

Advanced

Forecast future values and detect anomalies in temporal data using statistical and deep learning approaches.

Data Visualization

Beginner-Advanced

Create compelling visual narratives from complex datasets. Judged on clarity, insight, and design aesthetics.

Ethical AI

Intermediate

Develop solutions that address fairness, bias mitigation, and explainability in machine learning models.
