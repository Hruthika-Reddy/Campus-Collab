Course Description

This course introduces the language of engineers through technical drawing. Students learn to communicate design concepts using both manual drafting techniques and computer-aided design (CAD) tools.

Course Content

Unit 1: Fundamentals of Engineering Drawing

  • Drawing instruments and their uses
  • Lettering and dimensioning standards (BIS/ISO)
  • Geometrical constructions
  • Scales and their applications

Unit 2: Orthographic Projections

  • First angle vs third angle projection
  • Multiview drawings
  • Auxiliary views
  • Sectional views

Unit 3: Isometric and Perspective Projections

  • Isometric scale and isometric drawings
  • Conversion of orthographic to isometric views
  • Perspective drawing fundamentals

Unit 4: Computer-Aided Drafting (CAD)

  • Introduction to AutoCAD
  • Basic 2D commands and tools
  • Layer management
  • Creating and editing technical drawings

Important Tips

  • Always use proper drawing instruments (never sketch freehand)
  • Maintain consistent line weights (thick for visible edges, thin for construction)
  • Practice dimensioning standards regularly
  • Start learning AutoCAD shortcuts early
  • Keep your drawing sheets clean and smudge-free
